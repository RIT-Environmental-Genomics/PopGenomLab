# Lab 9: Population Genomics
Introductory tutorial for students enrolled in Environmental Genomics (BIOL-540/640) focused on Population Genomics analysis in R.

To get started, download the .Rmd file in the repository. Make sure that you have R and RStudio installed on your computer so you can open and run the code within the .Rmd file.

Code (in the form of Rmd file) for simulating a population genomic dataset and performing basic analyses and plots in R related to genetic diversity, population structure, selection, and isolation by distance.
